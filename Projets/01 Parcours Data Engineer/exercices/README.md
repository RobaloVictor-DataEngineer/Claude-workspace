# Exercices

Entraînement quotidien, séparé des projets pour ne rien mélanger.

- `python/` : scripts d'entraînement Python (pandas, OOP...)
- `sql/`    : requêtes d'entraînement (SQLBolt, pgexercises, DataLemur...)
- `java/`   : petits programmes Java (survol)

Règle : un fichier par notion ou par exercice, nommé clairement (ex : `sql/03_window_functions.sql`).
