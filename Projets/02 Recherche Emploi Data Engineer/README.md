# 02 — Recherche Emploi Data Engineer

## Objectif
Centraliser toute ma recherche d'emploi data engineer : Rouen / Île-de-France,
fourchette 40–50 k€, secteur pharma/santé en priorité, objectif avant septembre 2026.

## Arborescence (et quoi mettre où)
- **CV/** : mes versions de CV (par poste, par langue).
- **Lettres de motivation/** : lettres envoyées + modèles réutilisables.
- **Offres d'emploi/** : les offres qui m'intéressent (PDF, captures, liens) — la matière brute.
- **Suivi des candidatures/** : mon tableau de suivi (entreprise, poste, date, statut, relance).
- **Entreprises ciblées/** : fiches sur les boîtes visées (pharma/santé en tête).
- **Préparation entretiens/** : questions courantes, réponses STAR, notes par entreprise.
- **Portfolio et projets/** : liens GitHub et descriptions de mes projets (ETL...) à montrer.
- **Réseau et contacts/** : contacts LinkedIn, relances, événements.
- **Documents administratifs/** : diplômes, attestations, pièces d'identité.
